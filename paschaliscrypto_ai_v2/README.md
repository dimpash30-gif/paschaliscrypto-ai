# PaschalisCrypto AI v1

Πρώτη δημόσια έκδοση σε Next.js + Supabase.

## Τρέξιμο τοπικά
```bash
npm install
npm run dev
```
Άνοιγμα: http://localhost:3000

## Supabase
1. Φτιάξε δωρεάν project στο Supabase.
2. Πάρε URL και anon key.
3. Αντέγραψε `.env.example` σε `.env.local`.
4. Συμπλήρωσε τα στοιχεία.

## Public hosting
Ανεβαίνει δωρεάν σε Vercel.

## Σημείωση
Τα σήματα είναι demo στην v1. Το επόμενο βήμα είναι σύνδεση με τον Python scanner.
