# PaschalisCrypto AI v3

Έκδοση με Bybit public market data για 50 κρυπτονομίσματα.

## Deploy
Root Directory στο Vercel: `paschaliscrypto_ai_v3`

## Περιλαμβάνει
- Bybit ticker data από `/v5/market/tickers?category=linear`
- 50 USDT perpetual / linear ζεύγη
- Dashboard με Signals, Market και Statistics
